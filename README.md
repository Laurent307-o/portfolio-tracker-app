# Portfolio Tracker

Suivez vos portefeuilles boursiers (PEA, CTO, AV, PER) en temps réel.

## Features
- 📸 Scan IA de captures d'écran courtier
- 📊 Graphiques de performance
- 🔔 Alertes de cours personnalisées
- 📧 Rapport quotidien IA
- 🔒 Données sécurisées (Supabase + RLS)

## Tech Stack
- React + Vite
- Supabase (Auth + PostgreSQL)
- Yahoo Finance API
- Vercel (Hosting + Cron)

## Setup
1. Clone the repo
2. `npm install`
3. `npm run dev`
4. Configure Supabase credentials in `src/supabaseClient.js`
